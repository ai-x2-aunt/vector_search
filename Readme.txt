### 1. 설치 방법
1. **의존성 설치**
    - 프로젝트 루트 디렉토리에 `requirements.txt` 파일이 있는지 확인합니다.
    - 다음 명령어를 실행하여 필요한 패키지를 설치합니다:
        `pip install -r requirements.txt`

---
### 2. 환경 설정
1. **`.env` 파일 설정**
    - 프로젝트 루트 디렉토리에 `.env` 파일을 생성합니다.
    - OpenAI API 키를 포함시킵니다.
    - **예시 `.env` 파일:**
        `OPENAI_API_KEY=your-openai-api-key-here`
    - **주의:** `.env` 파일은 민감한 정보를 포함하므로, 버전 관리 시스템에 추가하지 않도록 `.gitignore` 파일에 추가합니다.

---
### 3. 벡터 스토어 구축
벡터 스토어는 채용 공고 데이터를 임베딩 벡터로 변환하여 저장하는 역할을 합니다. 이를 통해 효율적인 유사도 검색이 가능합니다.
1. **벡터 스토어 구축 스크립트 작성**
    예시 스크립트: `build_vectorstore.py`
### 4. 검색 및 요약 실행
검색 질의를 통해 벡터 스토어에서 유사한 문서를 검색하고, 이를 요약합니다.
1. **검색 및 요약 스크립트 작성**
    예시 스크립트: `search_and_summarize.py`
